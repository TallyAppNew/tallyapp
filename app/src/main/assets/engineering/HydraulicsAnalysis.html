Nomenclature

AV               = annular velocity (ft/min)

Q                 = circulate rate (gpm)

Dh               = hole diameter (inch)

Dp               = pipe or collar OD (inch)

MW              = mud weight (ppg)

N1 ; N2 ; N3 = jet nozzle size (32nd in.)

Pb               = bit nozzle pressure loss (psi)

HHP             = hydraulic horse power at bit

Vn               = jet velocity (ft/sec)

IF                = impact force (lb)

IF/sq in.       = impact force lb/sq in. of bit diameter
